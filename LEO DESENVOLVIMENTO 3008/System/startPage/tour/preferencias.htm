Visando maior comodidade ao usuário do Promob, existem diversas configurações que podem ser ajustadas para se tornarem padrões. O Promob possui configurações de preferência que controlam a aparência geral da interface do usuário, além de opções relacionadas a recursos específicos, como cores padrão de fundo, opções de ferramentas, e conversões de sistema de medidas.
